Ejemplos de Unidades de Medida en HTML y CSS

Unidades Absolutas

cm: Este cuadro tiene un ancho de 5cm.

mm: Este cuadro tiene un borde de 2mm.

px: Este texto tiene un tamaño de fuente de 20px.

Unidades Relativas

em: Este texto tiene un tamaño de fuente de 2em.

rem: Este cuadro tiene un margen de 1rem.

%: Este cuadro ocupa el 50% del ancho del contenedor.

vw: Este cuadro tiene un ancho de 10vw.

vh: Este cuadro tiene una altura de 20vh.
